볼륨 마운트 실습

이 파일은 호스트의 html/ 디렉토리에서 컨테이너의 /usr/share/nginx/html/로 마운트되었습니다.

실행 명령어:

docker run -d -v $(pwd)/html:/usr/share/nginx/html:ro -p 8081:80 nginx

실습 과제

  1. 이 파일을 수정해보세요
  2. 브라우저를 새로고침하여 변경사항을 확인하세요
  3. 컨테이너를 재시작해도 변경사항이 유지되는지 확인하세요

도커 명령어 연습

  • docker exec -it container-name ls /usr/share/nginx/html
  • docker logs container-name
  • docker stats container-name
